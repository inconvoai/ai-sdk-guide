# AI SDK Guide - Inconvo Data Agent Integration

A Next.js application demonstrating integration with [Inconvo](https://inconvo.ai)'s data agent tool using the Vercel AI SDK. This guide shows how to build interactive data visualization components that render tables and charts generated by AI.

## What's Inside

This project demonstrates:

- **Data Agent Integration**: Uses Inconvo's `dataAgent` tool to generate data insights from natural language queries
- **Custom Visualization Components**: Built-in components for rendering AI-generated data:
  - `InconvoTable`: Displays tabular data with proper formatting
  - `InconvoChart`: Renders interactive charts using Vega-Lite
  - `InconvoToolResult`: Wrapper component for handling tool results
- **Streaming AI Responses**: Real-time streaming of AI responses with tool calls
- **Type-safe Implementation**: Full TypeScript support throughout

## Getting Started

### Prerequisites

- Node.js 18+ installed
- pnpm package manager
- An Inconvo API key (set in `.env.local`)
- An AI gateway API key

### Installation

1. Install dependencies:

```bash
pnpm install
```

2. Copy `.env.example` to `.env.local` and fill in your API keys:

```bash
cp .env.example .env.local
```

Then edit `.env.local` with your actual API keys.

3. Run the development server:

```bash
pnpm dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## How It Works

1. **User Query**: User asks a data-related question (e.g., "Show me sales data for Q4")
2. **AI Processing**: The AI SDK streams the response and identifies when to call the Inconvo data agent
3. **Data Generation**: Inconvo's data agent generates appropriate tables or charts
4. **Visualization**: Custom React components render the results in an interactive format
